# Control de Caja Chica Diario Gratis

Aplicación web gratuita para llevar el **control de caja chica diaria** de tu negocio. Registra entradas y salidas de efectivo con balance automático en tiempo real, ideal para tenderos, emprendedores y pequeños comercios.

## ✨ Características

- **Registro de entradas y salidas** de efectivo con un clic.
- **Balance automático** actualizado al instante.
- **Reseteo diario** automático: cada día empieza en cero y se guardan los movimientos del día.
- **Persistencia local** en LocalStorage (privado y seguro).
- **Diseño Mobile-First** optimizado para Core Web Vitals.
- Sin registro ni instalación: funciona directamente en el navegador.

## 🚀 Uso

1. Selecciona si es **Entrada** o **Salida**.
2. Ingresa el monto y una descripción opcional.
3. Haz clic en **"Agregar Movimiento"**.
4. Revisa el balance al instante en la parte superior.

> Los datos se guardan únicamente en tu navegador (LocalStorage) y se reinician automáticamente al iniciar un nuevo día.

## 🛠️ Tecnologías

- HTML5 semántico
- Tailwind CSS (vía CDN)
- Vanilla JavaScript
- LocalStorage para persistencia
